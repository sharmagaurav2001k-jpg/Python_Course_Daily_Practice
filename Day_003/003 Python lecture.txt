Topic 5: Data Types Kya Hote Hain?
===============

- variable ke right side jo value hoti hai use data kahte hai, 
     or vah kis type ka data hai yah janne ke liye "type()" ka use kiya jata hai.


Data type : String, integer, Float, Complex, boolean, non

----------------
String :
-----------

- SAisa data type jo text and values dono ko store karta hai.
     Iska use un values ke liye hota hai jisme words, letters, sentences, symbols ya numbers text ke form mein hote hain.
     String value ko hamesha single quotes (' ') ya double quotes (" ") ke andar likha jaata hai.


Example 1:

          name = "Rahul"
          city = "Ahmedabad"
          course = "Python"
          mobile = "9876543210"
          message = "Welcome to Python Class"

# Text wali value ho to STRING.
# Number ko text form mein store kiya ja sakta hai. but iske liye questes mendetory hai.
# String ke andar alphabets, numbers, spaces aur symbols bhi store ho sakte hain.

Example 2:

          first_name = "Rahul"
          last_name = "Sharma"
          full_name = first_name + " " + last_name

          print(full_name)


Output:
          Rahul Sharma


# Yaha par first_name = "Rahul" aur last_name = "Sharma" tha.
# Phir hume ek variable diya gaya full_name name ka aur usme first_name + " " + last_name store kiya.
# Yaha + operator ka use do strings ko join karne ke liye kiya gaya.
# " " ka use beech mein space dene ke liye kiya gaya.
# Jaise hi humne system run kiya, dono strings join ho gayi.
# Isliye output mein "Rahul Sharma" print ho gaya.

# Agar hum text value ko quotes ke bina likhenge, to Python usse variable samjhega aur error dega.


Note : String ko hamesha quotes ke andar likhna zaruri hota hai. and Vriable ko kabhi bhi questes ke andar mat likhna.

     Yeh numbers positive, negative, ya zero koi bhi ho sakte hain.


# Decimal (.) na ho to INTEGER

Example 1:
age = 20
marks = 95

count = 0

Example 2:
num1 = 15

total = num1 + num2

      print(total)

 Out put :

Float :
--------

- Float (float) = Aisa data type jo decimal (point wale) numbers ko store karta hai.
     Iska use un values ke liye hota hai jisme decimal (.) hota hai. Decimal (.) wala number ho to "FLOAT"
     Decimal ke saath positive ya negative dono value ho sakti hai.
     

     Example 1:
     price = 99.99
     temperature = 36.5
     height = 5.8
     balance = -250.75

     Example 2:
     num1 = 10
     num2 = 3
     result = num1 / num2

      print(result)

 Out put :
           3.3333333333333335

# Yaha par num1 = 10 aur num2 = 3 tha.
# Phir humne ek variable diya gaya result name ka aur usme num1 / num2 store kiya.
# Jaise hi humne system run kiya, 10 ko 3 se divide kiya gaya.
# Division ka result decimal mein aaya, isliye output FLOAT value (3.3333333333333335) print ho gayi.

------------------
Boolean:
-----------


- Boolean (bool) = Aisa data type jo sirf do values store karta hai: True ya False. 
     Iska use kebal do condition ke result ko represent karne ke liye hota hai.


# condition agar right hota hai to TRUE

# condition agar wrong hota hai to FALSE

Example 1:
          is_logged_in = True
          has_permission = False
          is_adult = True
          is_raining = False


Example 2:
          age = 20
          is_adult = age >= 18

          print(is_adult)

     Out put :
               TRUE

# Yaha par age = 20 tha phir humne ek variable diya gya
# is_adult name ka or usme ye declare tha ki age jo hai vo 10 se bada hai.
# jaise humne system run kiya chek huaa ki age 18 se bada hai ya nahi, condition true hone ke vajah se output mein TRUE print ho gaya.


Note : Data type topic ye samjhne ke liye use hota hai ki variable mein stored Value(Data) kis type ka hai - means vo string hai, integer hai, ya float hai.

Note : type() function ka use kar ke aap pata kar sakte hai kl variable string , float ya boolean kis prakar ka data type hai.

Example:

          x = 10
          y = 10.5
          z = "Hello"
          w = True

          print(type(x))    # <class 'int'>
          print(type(y))    # <class 'float'>
          print(type(z))    # <class 'str'>
          print(type(w))    # <class 'bool'>

Input Function :
================

          - Input function ek built-in function hai jo program ko user dwara enter kiya gaya data receive karne ki suvidha deta hai.
               Python mein input() program ko temporarily rok deta hai, user se keyboard ke through value enter karwata hai, aur us value ko "string" ke form mein return karta hai. Iska use tab hota hai jab program ko fixed value ki jagah user se provide ki gayi value chahiye hoti hai.


Example 1:

          name = input("Enter your name: ")

          print(name)

Output:

          Enter your name: Rahul
          Rahul

# Yaha par name variable banaya gaya.
# Phir input() function ka use karke user se name enter karwaya gaya.
# Jab user ne Rahul type kiya, to Rahul value name variable ke andar store ho gayi.
# Jaise hi humne print(name) run kiya, output mein Rahul print ho gaya.

Example 2:

          city = input("Enter your city: ")

          print("Your city is", city)

Output:

          Enter your city: Ahmedabad
          Your city is Ahmedabad

# Yaha par city variable banaya gaya.
# Phir input() function se user se city ka name enter karwaya gaya.
# Jab user ne Ahmedabad type kiya, to Ahmedabad value city variable ke andar store ho gayi.
# Phir print() function ke through output mein Your city is Ahmedabad print ho gaya.


Note: input() function user se value lene ke liye use hota hai.
          input() function hamesha user ki value ko by default string ke form mein leta hai.
          Agar user number bhi enter kare, tab bhi input() use string maanta hai.
